Tarea de Base de Datos - Bandas y DiscosEste proyecto es para la Actividad 6 de la clase. 

Aquí aprendí a usar SQL para consultar información de bandas de música y sus discos.  

¿Qué usé? 
Visual Studio Code: Para escribir el código.PostgreSQL: Para guardar la información en tablas.
SQLTools: La extensión para conectar VS Code con la base de datos.

¿Qué hay en la base de datos?
Tengo dos tablas principales:
Bandas: Tiene el nombre de los grupos y de qué país son (como Chile, Alemania o UK).
Bandas_discos: Tiene los nombres de los álbumes y el año en que salieron.

¿Qué problemas resolví?
En el archivo de consultas logré hacer lo siguiente:  Buscar discos de bandas que no son de Alemania.  
Encontrar el disco más nuevo de grupos ingleses.  
Contar cuántos discos tiene cada banda en total.  
Buscar discos que empiezan con la letra A o que tienen varias palabras.
